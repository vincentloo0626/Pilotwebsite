# Pilot 演讲培训课程网站

这是一个简洁优雅的演讲培训课程网站，采用现代化的设计风格。

## 功能特点

- 响应式设计，适配各种设备
- 现代化的 UI 设计
- 流畅的动画效果
- 清晰的课程介绍
- 学员反馈展示
- 视频资源管理系统

## 技术栈

- 前端：HTML5, CSS3, JavaScript
- 后端：Node.js, Express
- 数据库：MongoDB
- 文件上传：Multer

## 项目结构 

http://localhost:3000
